Portfolio
  • Created Minesweeper in C++: Github
    • This project allows the user to play the classic game of minesweeper. The game generates a grid where bombs are randomly placed into positons unknown to the player. The player can select spaces to check whether or not a bomb is present and place flags in spaces where they believe a bomb to be. In order to win, the player must place flags on all the bombs without accidentally opening a bomb.
